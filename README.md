# Sistem Informasi OurUMKM

## Latar Belakang

Dalam era digital ini, mahasiswa dari berbagai fakultas di UIN Walisongo memiliki potensi besar dalam dunia kewirausahaan. Untuk mendukung dan memfasilitasi pemasaran serta penjualan produk UMKM mahasiswa, OurUMKM hadir sebagai platform online yang memungkinkan mahasiswa untuk dengan mudah memasarkan dan menjual produk UMKM mereka.

## Tujuan

Tujuan dari proyek ini adalah menyediakan platform yang efisien bagi mahasiswa UIN Walisongo untuk memasarkan dan menjual produk UMKM mereka secara online. Sistem ini juga bertujuan memberikan kemudahan bagi admin dalam mengelola produk, pengguna, serta melihat laporan transaksi.

## Fitur

1. **Pendaftaran dan Login**
   - Pengguna dapat mendaftarkan akun baru.
   - Pengguna dapat login dengan akun yang sudah terdaftar.

2. **Pengelolaan Admin**
   - Admin memiliki akses tinggi untuk mengelola data pengguna.
   - Menambah, melihat, mengubah, dan menghapus data produk.
   - Melihat dan mengelola daftar pengajuan produk.
   - Melihat dan mengelola data pengguna website.

3. **Pengelolaan User (Penjual)**
   - Penjual dapat mengajukan produk untuk ditampilkan di halaman produk.
   - Melihat status pengajuan produk (menunggu, disetujui, ditolak).
   - Melihat riwayat transaksi penjualan.

4. **Alur Pengajuan Produk yang Jelas**
   - Penjual dapat mengikuti alur sistem yang terstruktur, mulai dari pengajuan produk hingga penjualan.

5. **Halaman Produk**
   - Menampilkan produk yang dijual oleh mahasiswa.
   - Detail produk termasuk deskripsi, harga, dan gambar.

6. **Bukti Transaksi**
   - Pengguna dapat melihat bukti transaksi sebagai referensi penjualan.

## Cara Install

1. Unduh file zip dari [tautannya](https://drive.google.com/file/d/1zNr6OHnqKjUy_LdPA-prFn-EDlGxqYKG/view?usp=drive_link).
2. Ekstrak file zip ke dalam folder htdocs di direktori C:\xampp.
3. Pastikan nama folder hasil ekstrak adalah "ourumkm" (lokasi: C:\xampp\htdocs\ourumkm).
4. Buka XAMPP, aktifkan modul Apache, MySQL, dan FileZilla.
5. Buka browser, masukkan URL http://localhost/phpmyadmin/.
6. Buat database baru dengan nama "ourumkm".
7. Pilih database "ourumkm" dan klik "Import".
8. Pilih file "ourumkm.sql" di folder db dalam folder ourumkm yang telah diekstrak.
9. Klik "Import"..

## Cara Menggunakan

1. Buka browser dan akses: localhost/ourumkm.
2. Jalankan program dengan login sebagai penjual, pembeli atau admin.
